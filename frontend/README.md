# Supplier Hunter

Este projeto é um frontend React + Vite preparado para deploy no Vercel.

## 🚀 Rodando localmente
```bash
cd frontend
npm install
npm run dev
```

Acesse em: [http://localhost:3000](http://localhost:3000)

## 🌍 Deploy no Vercel
O projeto já contém um `vercel.json` configurado para rodar corretamente.
